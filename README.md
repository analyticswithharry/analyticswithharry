### Experience

I have worked extensively on building intelligent systems that operate in real-time environments, focusing on both machine learning and full-stack engineering.

On the machine learning side, I have developed and deployed models for classification, ranking, anomaly detection, and forecasting. These systems were enhanced through advanced feature engineering, statistical modeling, and continuous experimentation. I have also implemented model monitoring, drift detection, and automated retraining pipelines to ensure long-term reliability in production.

In parallel, I have engineered scalable backend systems using Node.js, Express, and Go, exposing machine learning capabilities through REST and gRPC APIs. These services were designed for low-latency inference and high availability, often integrated with streaming systems for real-time processing.

As a web developer, I have designed and developed full-stack applications from scratch, including dynamic frontends using React, Next.js, and modern UI tooling. These applications include complex systems such as dashboards, admin panels, e-commerce platforms, and AI-powered tools, all built with performance and user experience in mind.

I have also worked on data engineering pipelines using distributed systems like Kafka, Spark, and Airflow, enabling scalable data ingestion, transformation, and training workflows across cloud environments.
