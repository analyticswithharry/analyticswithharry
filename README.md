AI/ML Full Stack Engineer | LLM Systems | Web Developer

I build end-to-end intelligent systems that combine machine learning, large language models, and scalable web applications into production-ready products. ---

I specialize in building machine learning systems for classification, ranking, anomaly detection, forecasting, and personalization. These systems are designed with production constraints in mind, including monitoring, drift detection, and automated retraining pipelines.

Alongside traditional ML, I work extensively on LLM-based systems, including RAG pipelines, vector database integration, and agentic workflows. These systems enable context-aware reasoning, multi-step tool usage, and memory-driven applications that extend beyond static model responses.

On the engineering side, I develop scalable backend systems using Node.js, Express, and Go, exposing ML and AI capabilities through REST and gRPC APIs designed for low-latency inference and high reliability.

I also build full-stack web applications using React, Next.js, and modern frontend tooling, creating dashboards, admin panels, e-commerce systems, and AI-powered interfaces with a focus on performance and usability.

My data engineering experience includes building distributed pipelines using Kafka, Spark, and Airflow, enabling large-scale ingestion, transformation, and training workflows across cloud environments.
